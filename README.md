# 🧠 Neuron

**AI-Powered Research & Workflow Assistant** — A smart CLI tool built with LangChain that helps you research topics, summarize articles, draft content, and build a personal knowledge base — all from your terminal.

## Tech Stack

- **Python 3.10+**
- **LangChain** — Chains, Agents, Memory, Output Parsers
- **OpenAI / Groq** — LLM Provider
- **Tavily** — Web Search
- **FAISS** — Local Vector Store for Knowledge Base
- **Rich** — Beautiful CLI Output

## 🚧 Work in Progress

This project is actively under development. Features and documentation will be updated as the project evolves.
